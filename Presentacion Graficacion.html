Arquitectura y Componentes Principales

Por qué es crucial la visualización

La visualización de datos es como un traductor universal: convierte números complejos en historias visuales que cualquiera puede entender.

  • Análisis de métricas: Rendimiento de sistemas en tiempo real

GRÁFICAS 2D PROFESIONALES

Tipos de Visualizaciones y Personalización

Gráficas de Líneas y Barras

Líneas (plot)

Ideal para series temporales, tendencias y evolución de métricas

plt.plot(x, y, 'r-', linewidth=2)
plt.xlabel('Tiempo')
plt.ylabel('Valor')

Barras (bar)

Comparación de categorías, frecuencias y cantidades discretas

plt.bar(categorias, valores)
plt.xticks(rotation=45)
plt.tight_layout()

Dispersión e Histogramas

Dispersión (scatter)

Correlaciones, distribución de puntos y outliers

plt.scatter(x, y, c='blue',
            alpha=0.5, s=50)

Histogramas (hist)

Distribución de datos, frecuencias y análisis estadístico

plt.hist(datos, bins=30,
         edgecolor='black')

VISUALIZACIONES 3D

Representación de Datos Multidimensionales

Implementación de Gráficas 3D

from mpl_toolkits.mplot3d import Axes3D
import matplotlib.pyplot as plt
import numpy as np

# Crear figura 3D
fig = plt.figure(figsize=(12, 8))
ax = fig.add_subplot(111, projection='3d')

# Generar datos
x = np.linspace(-5, 5, 100)
y = np.linspace(-5, 5, 100)
X, Y = np.meshgrid(x, y)
Z = np.sin(np.sqrt(X**2 + Y**2))

# Superficie 3D
surf = ax.plot_surface(X, Y, Z, cmap='viridis',
                       linewidth=0, antialiased=True)

# Personalizar ejes
ax.set_xlabel('Eje X')
ax.set_ylabel('Eje Y')
ax.set_zlabel('Eje Z')
ax.set_title('Superficie 3D - Función Sinusoidal')

# Agregar barra de colores
fig.colorbar(surf, shrink=0.5, aspect=5)

# Ajustar ángulo de vista
ax.view_init(elev=30, azim=45)

plt.show()

DASHBOARDS PROFESIONALES

Integración de Múltiples Visualizaciones

Ejemplo de Dashboard Completo

class DashboardMetricas:
    def __init__(self):
        self.colores = {
            'primario': '#2E86AB',
            'secundario': '#A23B72',
            'exito': '#73AB84',
            'advertencia': '#F18F01',
            'error': '#C73E1D'
        }

    def crear_dashboard(self, datos):
        fig = plt.figure(figsize=(16, 10))
        fig.suptitle('Dashboard de Métricas de Aplicación',
                    fontsize=18, fontweight='bold')

        # Crear grid
        gs = fig.add_gridspec(3, 3, hspace=0.3, wspace=0.3)

        # 1. Gráfica de líneas - Tráfico
        ax1 = fig.add_subplot(gs[0, :2])
        ax1.plot(datos['tiempo'], datos['trafico'],
                color=self.colores['primario'], linewidth=2.5)
        ax1.set_title('Tráfico en Tiempo Real')
        ax1.grid(True, alpha=0.3)

        # 2. Gráfica de barras - Errores
        ax2 = fig.add_subplot(gs[0, 2])
        ax2.bar(datos['tipos_error'], datos['cantidad_errores'],
               color=self.colores['error'])
        ax2.set_title('Errores por Tipo')

        # 3. Histograma - Tiempos de respuesta
        ax3 = fig.add_subplot(gs[1, 0])
        ax3.hist(datos['tiempos'], bins=30,
                color=self.colores['exito'], alpha=0.7)
        ax3.set_title('Distribución de Tiempos')

        # 4. Gráfica de pastel - Usuarios
        ax4 = fig.add_subplot(gs[1, 1])
        ax4.pie(datos['usuarios_region'],
               labels=datos['regiones'],
               autopct='%1.1f%%')
        ax4.set_title('Usuarios por Región')

        # 5. Heatmap - Actividad
        ax5 = fig.add_subplot(gs[1, 2])
        im = ax5.imshow(datos['matriz_actividad'],
                       cmap='YlOrRd', aspect='auto')
        ax5.set_title('Mapa de Calor - Actividad')

        # 6. Métricas KPI
        ax6 = fig.add_subplot(gs[2, :])
        ax6.axis('off')
        self._mostrar_kpis(ax6, datos['kpis'])

        plt.show()
